What goes in a <head> tag anyway?

A brief guide to specifying HTML metadata

Introduction

The <head> element is a container for metadata (data about data) and is placed between the <html> tag and the <body> tag.

HTML metadata is data about the HTML document. Metadata is not displayed.

Metadata typically define the document title, character set, styles, scripts, and other meta information.

The Cheat Sheet

Element Name Tag Description
Title <title> Provides the page's title.
The title appears on the browser's title bar or in the page's tab.
It also provides a title for favorites and for search results.
Style <style> Provides a space for inserting CSS styling rules for the given page.
This is OK for simple single-page sites, but styling should idealy be put in a separate CSS file.
External Ressource Link <link> Links the current page to external ressources.
  • The href attribute specifies the URL of the linked ressource.
  • The rel attribute specifies the relationship between the page and the external ressource.
    The most common is "stylesheet" for CSS files.
Metadata <meta> Provides information like the page's character set, author, and viewport, for either the browser or search engines.
  • For the browser: Character set, Viewport, Refresh rate
  • For search engines: Author, Keywords, Description
JavaScript <script> Used to define client-side JavaScript scripts for this page.
Note: To avoid cluttering the HTML and to following the "seperation of concerns" doctrine, it is preferable to write scripts in a seperate .js file.
To do this, add a src attribute as follows: <script src="scriptfile.js">
Base URL <base> Specifies the base URL and/or target for all relative URLs in the page.
Must have at least one of the following attributes:
  • href: Defines the base URL
  • target: Tells the browser how to open the URL (new tab, window, etc.)

For valid HTML5...

The following should always appear for an HTML page to be considered "valid" to HTML5 standards:

  • <title>
  • <meta charset="UTF-8"> or other character sets.
  • ... and don't forget to delcare your language in the opening <html> tag!!!
